📌 Overview
A menu-driven calculator built using the C programming language. This project performs basic arithmetic operations along with additional features like memory storage and history tracking, making it more advanced than a simple calculator.


✨ Features

➕ Addition

➖ Subtraction

✖️ Multiplication

➗ Division (with zero-division handling)
🧠 Memory Store & Recall

📜 Last 10 Calculations History

🔁 Continuous execution (loop-based menu system)

🛠️ Tech Stack

Language: C

Concepts Used:

Loops (while)

Conditional Statements (if / switch)

Arrays

User Input/Output
